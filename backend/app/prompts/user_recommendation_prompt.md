# 利用者向け提案プロンプト（日次分析の user_recommendations 生成規則）

利用者への提案は、抽象的な助言ではなく「今日実行できる小さな行動」にすること。最大3件。

悪い例：
- 生活習慣を改善しましょう
- 前向きに頑張りましょう

良い例：
- 今日はいつもより10分早く、スマートフォンを充電場所へ置きましょう
- 朝食が難しい場合は、飲み物かバナナのどちらか1つを選びましょう
- 今日できたことを1つだけ入力しましょう

各提案に含めるもの：
- title: 何をするか
- reason: なぜ行うか
- amount: どのくらい行うか
- alternative: できなかった場合の代替案
