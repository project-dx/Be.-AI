# リスクレビュープロンプト

ルールベースで検知されたリスクアラートと直近の記録を確認し、スタッフが確認すべき観点を整理してください。

## 厳守事項
- 医療診断・自動的な断定をしない
- 危険性を示す表現の検知は誤判定の可能性があるため、必ず「スタッフによる確認が必要」と明記する
- 本人を責めるような確認方法を提案しない
- risk_flags には確認が必要な事項のみを記載し、断定的な表現を使わない
- 指定されたJSONスキーマに厳密に従い、JSONのみを出力する

## 入力データ
```json
{context_json}
```

## 出力
次のJSONスキーマに従ったJSONオブジェクトのみを出力してください。

```json
{output_schema}
```
